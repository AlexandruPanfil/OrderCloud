# OrderCloud Full-Stack Solution

A unified e-commerce ecosystem built with **.NET**, featuring a centralized API, a Web frontend, and a native Android application.

## 🏗 Architecture
* **OrderCloud.API**: ASP.NET Core REST API handling backend logic and OrderCloud integration.
* **OrderCloud.Blazor**: Web storefront and management interface.
* **OrderCloud.Android**: Native mobile application for end-users.
